# AoC

My attempt at Advent of Code.

## Usage

```shell
# Zig
cd <AOC_YEAR>/zig/day<n> # Input file is found by cwd
zig run part<n>.zig
zig test part<n>.zig # Test execution time

# Python
uv sync
uv run <AOC_YEAR>/python/day<n>/part<n>.py

# JavaScript/TypeScript
bun i
bun <AOC_YEAR>/javascript/day<n>/part<n>.{js,ts}
```
